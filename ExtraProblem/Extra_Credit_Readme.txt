In this question, we need to find the next state from the current state S1.
So, two text files are created: Extra_Credit_Facts.txt and Extra_Credit_Ops.txt
The Operation is provided to us, and we write it down in Extra_Credit_Ops.txt.
The Preconditions are written in the Extra_Credit_Facts.txt, and the resultant
state is provided in effects column.
According to the given operation, new additions are done, and the instances on 
which the del is applied are removed in this state.
